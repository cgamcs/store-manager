BD: render - postgresql (renovacion cada 90 dias)

Dependencias:
 - sequelize: se utiliza para la conexion y manejo de la base de datos PostgreSQL
 - pg y pg-store: se requiere para que sequelize pueda interactuar con PostgreSQL
 - express: framework para crear el servidor y manejar las rutas
 - nodemon (dev): herramienta para desarrollo que reinicia el servidor automaticamente al detectar cambios en el codigo
 - ts-node (dev): permite ejecutar archivos TypeScript directamente sin necesidad de compilarlos previamente
